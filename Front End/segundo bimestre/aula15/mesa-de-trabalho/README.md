# Aula 15 - Mesa de trabalho

A atividade consiste em realizar uma requisição para uma API pública, capturar os dados da resposta e apresentá-los em tela.
        
## Instruções

1. Acessa a documentação da [API JSON PLaceholder](https://jsonplaceholder.typicode.com/guide/) para realizar a atividade:

    - 1.1. Realize uma requisição para consultar todos as tarefas do recurso `todos`.

    - 1.2. Realize um filtro, utilizando `parametros` abordado na aula anterior, para retornar apenas as tarefas do usuário de `id 1`.

    - 1.3. Liste todas as tarefas em tela:

        - 1.3.1. Selecione o elemento `<ul class="c-lista">`

        - 1.3.2. Crie um elemento `<li class="c-lista__item">` para cada tarefa retornada.

<br><br>
